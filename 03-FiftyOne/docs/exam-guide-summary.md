# Training Guide Overview

## Program Overview

**Name:** FiftyOne: Visual AI Data Quality for Everyone
**Platform:** PlanetCorps AI Academy
**Version:** 1.0.0
**FiftyOne Version:** 1.14.0
**Python Version:** 3.13

## What This Training Covers

This training teaches you how to use FiftyOne — an open-source platform for visual AI data quality — to explore, analyze, and improve the datasets that power computer vision systems. It's designed for four audiences simultaneously: business leaders, data analysts, technical builders, and MBA/MSBA students.

## Training Structure

The training is organized into 5 domains, each weighted by importance:

| Domain | Name | Weight |
|--------|------|--------|
| 1 | Computer Vision Fundamentals | 15% |
| 2 | FiftyOne Platform & Architecture | 20% |
| 3 | Data Loading & Management | 20% |
| 4 | Visual Data Exploration & Quality | 25% |
| 5 | Model Evaluation & AI Workflows | 20% |

**Total:** 5 curriculum modules, 5 domain assessments, integrated exam scenarios, 4 hands-on exercises, and a comprehensive glossary.

## Assessment Format

### Domain Assessments
- Multiple-choice questions (A-D)
- Scenario-based — each question presents a realistic situation
- Tests judgment and application, not memorization
- Full explanations provided after each answer

### Exam Scenarios
- Multi-question scenarios that cross domain boundaries
- Designed to test integrated understanding
- Mirrors real-world problem-solving situations

### Passing Standard
- **Target:** 80%+ on all domain assessments
- **Mastery:** 90%+ on all assessments and scenarios

## Technical Prerequisites

- Python 3.13 installed
- pip package manager
- Terminal access (Command Prompt, PowerShell, or Windows Terminal)
- Modern web browser (Chrome recommended)
- ~2GB disk space for FiftyOne + datasets
- Internet connection for initial dataset downloads

## How to Use This Training

1. Follow the Learning Path phases in order
2. Read each module before taking its assessment
3. Complete exercises hands-on — don't just read them
4. Review assessment explanations for wrong answers
5. Use the glossary as a reference throughout

## Key Principles

This training follows three principles from the PlanetCorps AI Academy:

1. **Accessible first, then deep.** Every concept starts with plain language before going technical. Non-technical learners can stop at the accessible layer and still gain real understanding.

2. **Real examples, not theory.** Every concept is grounded in a real-world use case from manufacturing, healthcare, retail, autonomous vehicles, or other industries.

3. **Process thinking.** Every module connects to Lean AI Sigma principles — because data quality isn't just a technical problem, it's a process problem.
